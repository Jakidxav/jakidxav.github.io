Home About CV ABQ Mapping Tool

I designed a web application that allows users to create different mapping applications in the Albuquerque, NM area. My hope is that people interested in social and environmental justice can use this tool to create different maps, even without prior coding experience. Each map is a coding template that users can download or create custom maps directly on the site.

You can find the code on GitHub here.

You can demo the web app here.

A data aggregator map for overlaying points and polygons on different basemaps.
