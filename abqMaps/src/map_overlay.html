About the Overlay Map

This overlay map is meant to be a data aggregator for Albuquerque, New Mexico and the surrounding region. We hope that this site can serve as a communal resource for data, but also as a place where people can make maps and explore different datasets. We think that spatial questions are better answered when paired with a visual component.

Our goal is to host as many open-source datasets in JSON format as possible (converting from GeoTIFF, KMZ, SHP, etc). If you have more data that you would like to contribute to this map, please feel free to reach out! We plan to add more data so that people can use this site to visualize and download relevant data in JSON format.

How to use the Overlay Map (toggle)

- Click and drag in the map to pan your view. Use the +/- buttons to zoom the view in or out.

- To add data layers or change the basemap, click the icon in the upper right of the map to open the popup menu. On the popup menu you can select the layers you want to add to the map or use the radio buttons to change the basemap.

- Click the icon to measure distances between places. This feature measures total distance, so adding more than two points to the map will show you their combined distance. Please note: to end a current distance calculation, you will need to double-click on the last point that you want measured. You can then start another distance caluclation on the same map or clear the map of all calculations by again clicking the icon

- Click the icon to download a .png file of the map you have created. Please note: if you are using the Police Incidents point data, the download feature is currently slow to the point of now working.
